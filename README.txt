INFORMAÇÕES:

Projeto desenvolvido no front end em angular utilizando typescript.
Rota da aplicação padrão modificada para comunicação com back end, para executar a aplicação é necessário ter o npm instalado e rodar o comando em terminal da raiz:
npm run start 

Para rodar o servidor back-end é necessário executar o arquivo CrudSpringApplication.java na ide de preferencia.

Foram utilizadas as bibliotecas PrimeNG e Bootstrap para o desenvolvimento da aplicação angular.
O conteúdo da aplicação back end desenvolvida adota a rota de serviço 8080 e utiliza o Banco de dados H2 integrado ao spring e hibernate.

As rotas utilizadas para o desenvolvimento foram:
Angular: localhost:4200
Spring: localhost:8080
Caso necessário modificar as rotas, necessário modificar o arquivo proxy.conf para correta identificação das informações.
